Primeiros passos:

De maneira resumida, o html em sí só, é onde fica todas as informações e conteúdo do site. E de modo mais detalhado, o HyperText Markup Language (HTML) é uma tecnologia de desenvolvimento web, tem como a principal caracteristica o uso de suas marcas ou as chamadas tags para adicionar um objetivo ou função de cada recurso de um site, desde títulos até o comportamento relacionados à outras tecnolgias como CSS e Scripts com o JavaScript por exemplo.

Marcas / Tags

As marcas é o fundamento do html, sem as marcas o html não funciona, a tecnologia com o nome HyperText Markup Language tem em seu próprio nome a citação de uso de marcas, mas indo ao que interessa, as marcas são textos para definir uma identidade a um conteúdo do site, é como se falassemos para a tecnologia "-Olha, essa parte é isso, então se comporte como tal" e como nós que mandamos se der a ordem de maneira certa e bem detalhada ele vai conseguir seguir a ordem de maneira mais precisa. A funcionalidade do "<" e do ">" (Sinal de maior que.. e menor que..) é separar as marcas dos outros conteúdos, as marcas ou qual quer coisa entre esses sinais não irão ser considerados como conteúdo visivel do seu web site, o conteúdo que está em rosa é um comando da marca, acima tem uma lista de comandos básicos e suas funções dentro do html.

A maioria das tags (marcas) em html vai ter um fechamento, apenas repetindo a tag que você abriu com uma barra no inicio. Por Exemplo:

  • < example > < /example >

Mas algumas outras tags não tem fechamento, então colocamos a barra no final da primeira tag, ou deixamos do jeito que ele está mesmo

  • < example/ >

    ou mais facil:

    < example >

Simbulos

Alguns simbulos não funcionam no documento html, se acaso estiver "bugado" seu texto no parágrafo como acentuações e caractéres da escrita brasileira, adicionamos dentro do head um meta charset="UTF-8" desta maneira:

  • < head >
    < meta charset="UTF-8" >
    < /head >
